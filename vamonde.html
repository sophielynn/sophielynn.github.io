Competitive Analysis: Learnings from Current Solutoins

There are some obvious patterns among current tools: mobile-based tool for creating simple content, or web-based tool for creating complex content. This is caused by characteristics of each platform.

For Vamonde at this typical time to engage more business partners, we choose to stick to the web media only to create relatively complex content.
